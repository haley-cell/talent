# CV Match & Dispatch Context

This workspace powers the dashboard screen where a user uploads or pastes a CV, defines the job target, and asks the assistant to prepare a match recommendation.

## Inputs

- Candidate CV file or pasted resume text.
- Target role, job description, or must-have skill list.
- Optional recruiter notes.

## Outputs

- Match score and confidence label.
- Evidence-backed summary.
- Gaps and follow-up questions.
- Dispatch recommendation for human approval.
