# CRM Pipeline Optimizer Context

This workspace powers the dashboard screen where a user imports a CRM export, maps key fields, and asks the assistant to find business development actions.

## Inputs

- CRM CSV or pasted deal rows.
- Field mapping for company, owner, stage, value, last activity, next step.
- Optional analysis window and target segment.

## Outputs

- Pipeline health summary.
- Stalled deal list.
- Missing follow-up or missing owner alerts.
- Suggested next-best actions.
