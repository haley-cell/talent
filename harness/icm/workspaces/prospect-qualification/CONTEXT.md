# Prospect Qualification Context

This workspace powers the dashboard screen where a user defines an ICP, imports prospects, scores fit, and captures approved records to CRM.

## Inputs

- ICP description.
- Source list from CSV, company research, manual URL, or pasted contact notes.
- Optional exclusion rules and priority markets.

## Outputs

- Fit score and reason.
- Missing data warnings.
- Outreach angle.
- CRM capture payload for approved prospects.
